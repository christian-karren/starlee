A durable browser memory

People consume useful ideas every day, but those ideas disappear when they are not captured in a durable and searchable form. Starlee keeps a local Markdown record so knowledge can compound over time.

This fixture is deliberately long enough for reader-mode detection. It verifies that the built browser sensor finds an article, offers a one-click save affordance, and emits a normalized public payload.

A useful capture system should require almost no ceremony. The browser already knows what the reader can lawfully see, while the local engine already knows how to preserve, index, retrieve, and cite that material later.

Keeping those responsibilities separate makes the privacy boundary legible. The sensor observes the rendered page and the engine owns durable storage. No hosted corpus or remote inference provider sits between them.

Once captured, the original source remains traceable through its canonical URL and metadata. Search can retrieve the relevant passage while the agent composes a useful answer with evidence rather than vague recollection.
